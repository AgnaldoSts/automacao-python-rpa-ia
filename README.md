# 🤖 automacao-python-rpa-ia

Projeto de automação inteligente com **Python**, **RPA**, **IA**, **Excel** e **Streamlit**.

---

## 🚀 Tecnologias utilizadas

- Python 3.11
- Streamlit
- Pandas
- OpenPyXL
- Selenium
- IA (ChatGPT API - OpenAI)
- PowerShell (agendamento)
- Excel (análise e relatórios)

---

## 🧠 Funcionalidades

- Preenchimento automático de dados com RPA
- Leitura de arquivos Excel com Pandas
- Geração de gráficos
- Envio automático de e-mails com relatórios
- Interface gráfica interativa com Streamlit
- Integração com IA para análise e respostas
- Agendamento automático via PowerShell
